Tags

                

                    

                        

                            
A utiliser dans les tuiles, les champs select multiples, avec du texte à l'intérieur ou
                                des pictogrammes.

                            
<ul class="tags list">
    <li class="tag>

<p class="tags">
    <a class="tag>

<ul class="tags list">
    <li class="tag tag_icon>
        <i class="fas fa-file-pdf"></i>

<span class="tag>
    <a class="button button_icon tight primary">
        <span class="button__icon">
            <i class="fal fa-times">
